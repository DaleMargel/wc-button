wc-button

Easy to use buttons

Features

  • single file, ready to go, import and use!
  • small size 2k to 3k w/o compression
    ...depending on whether styles are embedded
  • web component: compatible with all frameworks
  • control contains it's own design system
  • code is simple enough to change what you don't like
  • just import it and use it like a normal html tag
    ...no special skills needed!

The look of the button can be set through attributes

  • color: blank, default, primary, success, info, warning, danger
  • size: small, medium, large, huge
  • trim: square, smooth, round
  • span: narrow, normal, wide, extrawide, full
  • theme: light, dark
